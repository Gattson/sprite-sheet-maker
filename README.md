# Sprite Sheet Maker

A pixel-art and freeform-paint sprite sheet / animation editor that runs
entirely in your browser — nothing is uploaded, your art stays on your
machine.

**Use it here: https://gattson.github.io/sprite-sheet-maker/**

- Two project modes: classic pixel editor (grid, palettes, crisp pixels) and
  freeform paint (pressure-sensitive brushes, textures, smudge, layers with
  blend modes).
- Animation tools: frames, onion skinning, in-canvas playback, free
  transform of selections.
- Export: PNG sprite sheets, animated GIF / APNG, video, or a ZIP of frames.
  Projects save/load as JSON files.
